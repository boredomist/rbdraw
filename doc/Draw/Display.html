Draw::Display

This class handles the connection to the X server. Chances are you won’t need to touch this at all, since Draw::Window.new can be called without a Display parameter, and will default to the one you’ll most likely want to use anyway.

Creates a new Display object. Can be called with either one or no arguments to represent the name of the X server to connect to. With no args, it will connect to the default display
