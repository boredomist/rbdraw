rbdraw

DESCRIPTION:

rbdraw is a 2D drawing library for Ruby. It works by wrapping some standard X11 functions into an easier to work with way.

SYNOPSIS:

Check out the file ‘test.rb’ for some small examples of how to use rbdraw. More examples will be added as rbdraw gets more features.
